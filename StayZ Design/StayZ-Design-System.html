Feature Flows
Flow 1 — Tìm kiếm & Đặt phòng
Home → Tap search → Search screen
  → Nhập điểm đến + ngày + số khách → Apply filter
    → Empty / Offline state nếu không có kết quả hoặc mất mạng
    → Tap khách sạn → Hotel Detail
      → "Chọn phòng" → Room Selection
        → "Đặt ngay" → Chọn thời gian & khách
          → Checkout → Xác nhận thanh toán
          → Booking Confirmation (QR)
Flow 2 — Auth
App Launch → Check JWT token còn hạn?
  YES → Home
  NO → Splash → Onboarding (lần đầu)
    → Login → Success → Home
          → Fail → Hiển thị lỗi, giữ nguyên màn
    → "Đăng ký" → Register → OTP → Home
Flow 3 — Wishlist
Bất kỳ màn có hotel card → Tap ♡
  Chưa login → redirect Login
  Đã login → Toggle save (optimistic UI) → Toast
Bottom Nav "Yêu thích" → Danh sách → Hotel Detail
Flow 4 — Quản lý đặt phòng
Bottom Nav "Đặt phòng" → My Bookings
  Tab "Sắp tới" → Upcoming Booking Detail → Xem QR
    → "Hủy" → Confirm dialog → Cancel Booking Result
  Tab "Hoàn thành" → Completed Booking Detail → "Đánh giá" → Rating screen → Submit
  Tab "Đã hủy" → Cancelled Booking Detail → Đặt lại phòng này

VND Currency Format (Dart)
// Format: ₫1.800.000 hoặc ₫1.8M (compact)
String formatVND(int amount) {
  if (amount >= 1000000) {
    double m = amount / 1000000;
    return '₫${m.toStringAsFixed(m % 1 == 0 ? 0 : 1)}M';
  }
  return '₫${NumberFormat("#,###", "vi_VN").format(amount)}';
}
